Politica de privacidad de KePedimos

Ultima actualizacion: 13 de mayo de 2026

Esta politica explica como KePedimos accede, recopila, usa, comparte, protege y conserva datos de usuarios de la app KePedimos, el sitio web y la API asociada. Para consultas de privacidad, escribi a admin@kepedimos.com.

Datos que recopilamos

  • Datos de cuenta: nombre, email, rol de usuario, estado de cuenta y credenciales protegidas.
  • Datos de contacto y direccion: domicilios de entrega, referencias, localidad, provincia, codigo postal y coordenadas del pin de entrega.
  • Datos de pedidos: carrito, productos, comercio, importes, estado del pedido, metodo y estado de pago.
  • Datos de ubicacion: ubicacion aproximada o precisa para validar cobertura, completar direcciones y, en cuentas de repartidor, compartir el avance de una entrega activa.
  • Datos de repartidor o comercio, cuando correspondan: telefono, vehiculo, documentacion cargada, datos operativos y liquidaciones.
  • Tokens de notificaciones push y datos tecnicos necesarios para entregar mensajes sobre pedidos, cuenta y operaciones.
  • Archivos que subas voluntariamente, como imagenes de productos, comprobantes o documentacion operativa.

Ubicacion en segundo plano

KePedimos recopila datos de ubicacion (location) para habilitar seguimiento del reparto, actualizacion del pedido y ETA en background, incluso when the app is closed or not in use. Esta funcion se usa solo para repartidores durante entregas activas. La ubicacion no se usa para publicidad.

Como usamos los datos

  • Crear y proteger cuentas, autenticar usuarios y mantener sesiones.
  • Procesar pedidos, pagos, entregas, notificaciones, reclamos y soporte.
  • Mostrar comercios disponibles segun cobertura, direccion o ubicacion indicada.
  • Permitir a clientes, comercios y repartidores ver estados necesarios para completar el pedido.
  • Cumplir obligaciones legales, contables, antifraude y de seguridad.

Con quien compartimos datos

Compartimos solo los datos necesarios para prestar el servicio: comercios reciben datos del pedido; repartidores reciben datos necesarios para entregar; clientes ven informacion de seguimiento; proveedores de pago como Mercado Pago procesan pagos; proveedores de mapas, geocodificacion, rutas, hosting y notificaciones procesan datos tecnicos necesarios. No vendemos datos personales ni los usamos para publicidad comportamental.

Seguridad

Usamos HTTPS, tokens de sesion, almacenamiento seguro en la app para credenciales de sesion y controles de acceso por rol. Las credenciales sensibles de proveedores se almacenan cifradas en backend cuando corresponde.

Retencion y eliminacion

Conservamos datos mientras la cuenta este activa o mientras sean necesarios para prestar el servicio, resolver reclamos, cumplir obligaciones legales, contables o de seguridad. Podes solicitar o iniciar la eliminacion desde la app o desde la pagina de eliminacion de cuenta. Al eliminar una cuenta, borramos datos de perfil, direcciones, tokens push y otros datos personales asociados; algunos registros transaccionales pueden conservarse anonimizados cuando sea necesario por obligaciones legales o contables.

Tus controles

  • Podes editar o eliminar direcciones desde la app.
  • Podes cerrar sesion y revocar permisos de ubicacion o notificaciones desde los ajustes del dispositivo.
  • Podes eliminar tu cuenta desde la pantalla de perfil de la app.
  • Podes contactarnos en admin@kepedimos.com para consultas o solicitudes de privacidad.
